# 🚀 Task Management System

A full-stack Task Management Web Application that helps users efficiently organise, track, and manage daily tasks.

Built using modern web technologies, this project demonstrates full-stack development skills, including frontend, backend, and database integration.

---

## 📌 Features

✅ Create, update, and delete tasks  
✅ Mark tasks as completed  
✅ View all tasks in one place  
✅ RESTful API integration  
✅ Responsive user interface  
✅ Real-time data handling with backend  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS / Tailwind (if used)

### Backend
- Node.js
